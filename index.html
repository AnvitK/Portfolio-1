My Personal Projects

YelpCamp Website

This website lets the user to register and login. The users can view all the campgrounds. Only the registered users can add campground or edit their campgrounds and can also add or delete reviews on all the campgrounds. The registered user can add campground details like title, location, images, price and description of the campground. The added campground is then displayed in a show page where its location is also displayed on the map and is displayed as a card on the campgrounds index page.


Main Technologies used: Express, NodeJS, Mongoose, MongoDB, Passport, Cloudinary, MapBox, etc.

Hosting Technologies used: Mongo Atlas, GitHub, Render.com

Forkify Website

This website lets you search tons of recipes based on the keyword. The selected recipe is displayed with its preparation duration, default servings, recipe ingredents and also a button to know more about the recipe. The user can increase or decrease servings, which thereby changes recipe ingredents quantities. The user can also bookmark a recipe which gets displayed in bookmark tab. The user can also add a recipe.


Main Technologies used: JavaScript, API, Parcel, NodeJS, SCSS, etc.

Hosting Technologies used: GitHub, Netlify

Mapty Website

This mini project website gets your current location on the map. The user then selects the desire location, where the user selects the workout type, distance, duration, cadence or elevation gain based on workout type on the form. The workouts are then displayed in tiles from the local storage. On clicking on the particular workout tile, that's workout location is displayed on the map.


Main Technologies used: JavaScript, Leaflet, OOPs, LocalStorage, etc.

Hosting Technologies used: GitHub, Netlify
